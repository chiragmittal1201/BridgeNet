# BridgeNet – Centralized Alumni Management System

BridgeNet is a web-based platform designed to help colleges maintain a **verified alumni database** and enable structured engagement between students and alumni within an institution-controlled environment.

## Live Demo

https://alumni-management-240b6.web.app

## Problem Addressed

* Alumni data becomes scattered after graduation
* Students lack a trusted way to contact alumni
* Informal platforms (WhatsApp groups, etc.) lack verification and control

## Key Features

* Role-based system (Admin, Alumni, Student)
* Admin verification of alumni profiles
* Student directory of verified alumni
* Alumni–student engagement request system
* Contact details shared only after alumni approval

## Tech Stack

Frontend: React.js
Authentication: Firebase Auth
Database: Firebase Firestore
Hosting: Firebase Hosting

## Project Structure

```
src/      → React components and application logic  
public/   → Static assets
```
